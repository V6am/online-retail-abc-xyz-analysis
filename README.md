# ABC/XYZ-анализ онлайн-ритейла (Online Retail II)

[![Kaggle](https://img.shields.io/badge/Data-Kaggle-blue)](https://www.kaggle.com/datasets/mashlyn/online-retail-ii-uci)
[![License](https://img.shields.io/badge/License-MIT-green)](LICENSE)

Совмещённый **ABC/XYZ-анализ** товарного ассортимента британского интернет-магазина. Проект выполнен в рамках изучения методов сегментации продаж и прогнозирования спроса.

---

##  Цель проекта

Разделить товары на группы по двум критериям:

- **ABC** — вклад в выручку (принцип Парето)
- **XYZ** — стабильность продаж (коэффициент вариации)

Совмещённая матрица **ABC+XYZ** позволяет:
- выявить топ-товары (группы AX, AY, AZ),
- определить проблемные позиции (CZ, CY),
- оптимизировать складские запасы и ассортимент.

---

##  Исходные данные

**Источник:** [Online Retail II Data Set](https://www.kaggle.com/datasets/mashlyn/online-retail-ii-uci)

**Описание:** транзакции британского интернет-магазина (подарки, товары для дома) за период **декабрь 2010 – декабрь 2011**.

**Поля:**
- `InvoiceNo` — номер заказа
- `StockCode` — артикул товара
- `Description` — описание товара
- `Quantity` — количество
- `UnitPrice` — цена за единицу
- `InvoiceDate` — дата заказа
- `Customer ID` — идентификатор клиента
- `Country` — страна

---

##  Методология

### 1. ABC-анализ (по выручке)
Товары ранжируются по сумме `TotalPrice = Quantity × UnitPrice`:

| Группа | Доля в выручке | Доля товаров |
|:------:|:--------------:|:------------:|
|   A    |      ~80%      |     ~20%     |
|   B    |      ~15%      |     ~30%     |
|   C    |       ~5%      |     ~50%     |

### 2. XYZ-анализ (по стабильности)
Рассчитывается коэффициент вариации (CV) месячных продаж:

| Группа | Коэффициент вариации | Характер спроса   |
|:------:|:--------------------:|:------------------|
|   X    |        < 25%         | Стабильный        |
|   Y    |       25–50%         | Умеренно сезонный |
|   Z    |        > 50%         | Нерегулярный      |

### 3. Совмещённая матрица ABC+XYZ
Товары классифицируются по обеим осям (например, **AZ** — высокая выручка, но нерегулярный спрос).
